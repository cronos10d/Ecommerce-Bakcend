# Ecommerce Backend

Este es el backend de un sistema de comercio electrónico. Está diseñado para proporcionar una API RESTful para gestionar productos, usuarios y órdenes, cupones, carrito.  

## Proyecto

**Nombre del Proyecto**: Ecommerce Backend  
**Descripción**: API para gestionar productos, usuarios, órdenes, etc de un ecommerce, construida con Node.js.
**Tecnologías**: Node.js, Express, MongoDB, JWT, Mongoose, MysqlServer.

## Instalación

1. **Clona el repositorio:**

   ```bash
   git clone https://github.com/cronos10d/Ecommerce-Bakcend.git
   cd Ecommerce-Bakcend


2. **Instala las dependencias:**

  npm install

3. **Configura las variables de entorno en un archivo .env con:**
   
    PORT=5000
    MONGO_URI=tu_url_de_conexion_a_mongodb
    JWT_SECRET=tu_clave_secreta

4. **Ejecuta el servidor:**

   npm start
